# Tines

Tines is a Dublin-based automation platform that has become a cornerstone of the **Agentic AI** ecosystem in Europe. Originally focused on security orchestration (SOAR), Tines now enables enterprises to build and deploy autonomous agents for complex technical workflows.

## 🚀 Strategic Overview
- **Agentic Pivot (2025):** Launched "Tines Agents," moving from static "drag-and-drop" workflows to dynamic agents that use LLMs to interpret logs and execute remediation steps autonomously.
- **Enterprise Scale:** Used by global leaders like Canva, Snowflake, and McKesson to automate high-frequency security tasks.
- **Market Positioning:** Positions itself as the "glue" for the modern AI-powered enterprise, orchestrating tools from Microsoft, AWS, and CrowdStrike.

## 🛠️ Key Milestones (2025–2026)
- **2025:** Reached $100M ARR milestone; expanded Dublin R&D headquarters.
- **2026:** Launched "Tines Hub," a marketplace for pre-trained agentic workflows across security, IT, and HR.
- **Global Presence:** Strong growth in the US market, with a secondary HQ in Boston.
