# Wayve

Wayve is a London-based pioneer in **Embodied AI** for autonomous driving. Founded by Cambridge engineers, it uses end-to-end deep learning to allow vehicles to navigate complex urban environments using only cameras and sensors.

## 🚀 Strategic Overview
- **Massive Backing:** Secured $1.05B in Series C funding (2024) led by SoftBank, NVIDIA, and Microsoft.
- **AV 2.0:** Unlike traditional AVs that rely on HD maps and rule-based logic, Wayve uses a "General Purpose" driving model that learns from data, similar to how LLMs learn language.
- **Physical AI Leadership:** Recognized as one of the few companies globally capable of deploying "World Models" in high-speed physical environments.

## 🛠️ Key Milestones (2025–2026)
- **2025:** Expanded "Embodied AI" research lab in London; established US testing presence.
- **2026:** Launch of London's first commercial-scale autonomous ride-hailing trials using its 2.0 software stack.
- **Hardware Agnostic:** Partnered with major OEMs to embed Wayve's "brain" into production vehicles.
